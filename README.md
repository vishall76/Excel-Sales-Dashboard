# Excel Sales Dashboard

## 📌 Overview
This project is an interactive Excel Sales Dashboard designed to analyze and visualize sales performance. It provides insights into sales trends, profit, customer performance, and regional analysis using dynamic Excel features.

## 🚀 Features
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Slicers & Timeline Filters
- KPI Cards
- Sales Trend Analysis
- Region-wise Sales Analysis
- Category-wise Performance
- Dynamic Reports

## 🛠️ Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline
- Conditional Formatting
- Excel Functions (SUMIFS, IF, VLOOKUP/XLOOKUP, etc.)
